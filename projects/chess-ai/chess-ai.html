Chess AI

GitHub Repository: Chess-AI

I started this independent project as a way to teach myself about game theory and artifical intelligence, and gain a deeper understanding of algorithm design. Chess is a two-player deterministic zero-sum game of perfect information, which makes it a great way to explore applications of the minimax algorithm, alpha-beta pruning, and reinforcement learning. I plan on using this page to share some of the challenges I face while creating this AI; I would also like to use this page to share other interesting thoughts about this project as they occur to me.
